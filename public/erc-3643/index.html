ERC-3643: compliance programable para activos tokenizados

ERC-3643 permite emitir tokens con identidad, elegibilidad y restricciones de transferencia, una base útil para security tokens y proyectos RWA.

Por qué importa

En activos regulados no alcanza con transferir libremente. El token debe saber si una wallet está verificada, si una jurisdicción está permitida y si la operación cumple reglas definidas.

Componentes típicos

  • Identidad digital o registro de inversor.
  • Compliance contract con reglas de transferencia.
  • Agentes autorizados para aprobar, pausar o administrar.
  • Eventos auditables para monitoreo post-emisión.

Cuándo usarlo

ERC-3643 es especialmente relevante cuando el token representa derechos económicos, deuda, participación o instrumentos que no deben circular fuera de un universo de inversores habilitados.

TOKAI RWA usa estándares tokenizados como herramientas de control operativo, no como atajos para evitar compliance.
